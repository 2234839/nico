中文文档

也许你已经安装好了 nico，但是我还是得重复一下。使用 npm 安装 nico 很简单，只需要在终端里输入：

$ npm install nico -g

注意，一定要有 -g 哦，这样才能够在终端里调用到 nico。

Windows 用户请注意，安装可能不会一帆风顺，在安装 nico 之前，请确认你的环境：

Windows XP/Vista/7 需要安装：

  • Python 2.7.3
  • Microsoft Visual Studio C++ 2010 (Express version)
  • 如果你是 64 位的Windows 7，还需要安装 Windows 7 64-bit SDK

Windows 8 需要安装：

  • Python 2.7.3
  • Microsoft Visual Studio C++ 2012 for Windows Desktop Express version

参考 https://github.com/TooTallNate/node-gyp
